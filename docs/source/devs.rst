Contributing and Developing
===========================

HTMap is open to contributions!
Please feel free to submit a
`Pull Request <https://github.com/htcondor/htmap/pulls>`_ on GitHub.
If you contribute to HTMap, please add your name to the
``CONTRIBUTORS`` file in the repository root (if you want to be listed).
